# Simplex Assembler & Emulator

A C++ implementation of a two-pass assembler and emulator for the Simplex Instruction Set Architecture (ISA).

## Features

- Two-pass assembly process
- Symbol table generation
- Label resolution
- Machine code generation
- Instruction decoding
- Arithmetic instructions
- Branch instructions
- Memory operations
- Error handling and logging

## Assembler

### Pass 1
- Builds symbol table
- Resolves labels

### Pass 2
- Generates machine code

## Emulator

Simulates:
- Registers
- Memory
- Program Counter (PC)

and executes the generated machine code instruction-by-instruction.

## Example Programs

- Fibonacci Sequence
- GCD Computation
- Bubble Sort
- Sum of Digits

## Technologies Used

- C++
- STL
